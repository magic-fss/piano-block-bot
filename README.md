## 🚀 v2.0.0 核心重构

### 吸附机制架构
- **检测区域**：仅扫描区域 **顶部 1/4（50px）**，黑块刚进入检测框即触发
- **点击位置**：砸在区域 **下半部偏下（3/4 处，150px）**，利用游戏纵向容错/吸附机制提前消除黑块
- **触发阈值**：`MIN_BLACK_PIXELS` 从 50 降至 **15**，黑块露头即满足条件

### 性能优化
- **删除 `break`**：一帧内可连续处理多列，消除高速模式下的漏点暴毙
- **移除实时日志**：运行时零打印，消除高频函数调用开销；仅在暂停/停止时输出统计
- **零延迟主循环**：运行时去掉 `sleep`，检测频率达到物理上限；暂停时才让出 CPU

## 🎮 使用方法

### 1. 启动游戏

打开 **钢琴块2**（窗口标题必须为 `钢琴块2`），进入任意关卡。  
注：也可修改源码中 `GAME_WINDOW_TITLE` 匹配其他窗口标题。

### 2. 运行脚本

```bash
python piano_block_tui.py
```

> ⚠️ **必须以管理员身份运行**，否则全局热键（F10/F12/ESC）无法注册。

### 3. 调整检测区域

脚本启动后弹出 4 个带彩色边框的半透明窗口：

| 颜色 | 对应列 |
|---|---|
| 🔴 红色 | 第 1 列 |
| 🟢 绿色 | 第 2 列 |
| 🔵 蓝色 | 第 3 列 |
| 🟡 黄色 | 第 4 列 |

**拖拽调整**：鼠标按住框内任意位置拖动，对齐游戏对应列。

**调整原则（v2.0.0）**：
- 框的**顶部**对齐游戏黑块开始出现的位置（检测在此触发）
- 框的**底部**向下延伸，确保 3/4 处落在游戏可点击的安全白区（吸附落点）
- 框的**宽度**略大于单列宽度即可
- 框的**高度**固定为 200px，不建议随意缩小（需要保证底部有足够的安全点击纵深）

## ⚙️ 配置说明

脚本顶部「终极速度配置」区：

```python
# 区域尺寸
AREA_WIDTH   = 150   # 检测框宽度（像素）
AREA_HEIGHT  = 200   # 检测框高度（像素），v2.0.0 固定 200，提供底部吸附纵深
AREA_GAP     = 50    # 列间距（像素）

# 黑块检测阈值
GRAY_TOLERANCE   = 6   # 灰度容忍度（0~255），越小越严格
MIN_BLACK_PIXELS = 15  # 触发点击的最小黑像素数，v2.0.0 极限降低

# 热键
START_HOTKEY            = 'f10'
STOP_HOTKEY             = 'f12'
EMERGENCY_STOP_HOTKEY   = 'esc'
```

### 高速场景调参建议

| 场景 | MIN_BLACK_PIXELS | 说明 |
|---|---|---|
| 标准模式 | 15 | 默认，碰顶即点，吸附消除 |
| 黑块较小/较淡 | 10~12 | 更早触发，误触风险微增 |
| 背景偏暗/干扰多 | 20~25 | 保守策略，避免误触白块 |

> v2.0.0 的核心速度来自**检测范围缩小**（顶部 50px）和**吸附点击**（3/4 处），而非单纯压低阈值。如遇到误触，优先检查框的位置是否对齐，而非盲目降低阈值。

## ⚠️ 注意事项

1. **管理员权限**：`keyboard` 库注册全局热键需要管理员权限，否则热键无效。
2. **关闭鼠标加速**：控制面板 → 鼠标 → 指针选项 → **取消勾选「提高指针精确度」**，避免 Windows 鼠标加速干扰点击精度。
3. **窗口标题匹配**：脚本通过窗口标题 `"钢琴块2"` 定位游戏，请确保游戏窗口标题一致。
4. **单显示器**：当前版本基于绝对屏幕坐标，多显示器环境下请确保游戏在主显示器运行。
5. **吸附机制依赖**：v2.0.0 假设"点击黑块所在列的偏下位置可消除黑块"。如实际测试中发现 3/4 处偶尔无效，将 `click_y = y + _h * 3 // 4` 中的 `3 // 4` 适当减小（如 `2 // 3` 或 `1 // 2`），检测部分（顶部 1/4）无需改动。
6. **CPU 占用**：运行时 Python 进程会吃满一个 CPU 核心，属零延迟循环的取舍。如机器散热敏感，可将主循环 `else: time.sleep(0.05)` 改为 `0.01` 稍作妥协。

## 🛠️ 技术栈

- **截图**：`mss`（跨平台极速截图，合并包围盒一次抓取 4 列）
- **图像处理**：`numpy` 布尔掩码（跳过 OpenCV 冗余遍历，直接取 B 通道统计黑像素）
- **窗口操作**：`pygetwindow` + `pywin32`（SetCursorPos + mouse_event 零延迟点击）
- **热键监听**：`keyboard`（全局热键）
- **GUI**：`tkinter`（可视化拖拽区域）
